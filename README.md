# Aplikasi lab bahasa atau listening

Dibuat dengan : 
- PHP 7.3 
- Template admin themekit

Fitur aplikasi : 
- Login dan logout 
- Mengelola data soal 
- Mengelola data kelas 
- Mengelola data bahasa 
- Memilih soal aktif 
- Tampil soal sesuai soal aktif

Akses : 
- Halaman utama : localhost/listening
- Halaman admin : localhost/listening/admin
